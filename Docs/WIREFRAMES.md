## Состояния элементов

### Кнопка "Проверить"
```
idle:     [Проверить]
hover:    [Проверить]  (фон темнее)
loading:  [o Проверка...]
disabled: [Проверить]  (серый, нет текста)
```

### Карточка ошибки
```
idle:     | "ошибка" -> "исправление"  [Применить]|
hover:    | "ошибка" -> "исправление"  [Применить]| (тень)
applied:  | v Исправлено                          | (зелён)
dismissed:| (скрыта из списка)                    |
```

### Переключатель языка
```
idle:     {Авто (RU) v}
open:     +----------+
          | * Авто   |
          | o Русский|
          | o English|
          +----------+
```

### Тумблер приватности
```
off:      o----- Не отправлять текст
on:       -----o Не отправлять текст  [!] LT и плагиат OFF
```

## Страница Политики

```
+------------------------------------------------------------------------------+
|  TextCheck                                        [<- На главную]            |
+------------------------------------------------------------------------------+
|                                                                              |
|  Политика конфиденциальности                                                 |
|  ============================================================                |
|                                                                              |
|  Последнее обновление: ноябрь 2025                                           |
|                                                                              |
|  1. Какие данные мы обрабатываем                                             |
|  --------------------------------                                            |
|  * Текст, который вы вводите для проверки                                    |
|  * Язык текста (RU/EN)                                                       |
|  * Ключевые слова для SEO-анализа                                            |
|                                                                              |
|  2. Как обрабатывается текст                                                 |
|  ---------------------------                                                 |
|  * Орфография/грамматика: отправляется на сервер LanguageTool                |
|  * Плагиат: фрагменты отправляются в поисковые системы                       |
|  * Остальные проверки выполняются локально в браузере                        |
|                                                                              |
|  3. Режим приватности                                                        |
|  ---------------------                                                       |
|  При включении "Не отправлять текст":                                        |
|  * Проверка LanguageTool отключается                                         |
|  * Проверка плагиата отключается                                             |
|  * Работают только локальные проверки                                        |
|                                                                              |
|  4. Хранение данных                                                          |
|  ------------------                                                          |
|  * Текст сохраняется только в localStorage вашего браузера                   |
|  * Сервер не хранит ваши тексты                                              |
|  * LanguageTool не сохраняет проверенные тексты                              |
|                                                                              |
|  5. Контакты                                                                 |
|  ----------                                                                  |
|  По вопросам: example@textcheck.app                                          |
|                                                                              |
+------------------------------------------------------------------------------+
|  (c) 2025 TextCheck                                                          |
+------------------------------------------------------------------------------+
```
